# Files Moved to Python Directory

This directory contains the Python implementation of Ralph, separated from the original bash/Amp implementation.

## Files Included

- `ralph.py` - Main Python implementation
- `requirements.txt` - Python dependencies
- `Makefile` - Build and development commands
- `.gitignore` - Git ignore rules for Python project
- `README.md` - Quick reference for this directory
- `ralph_python_README.md` - Full documentation
- `QUICKSTART.md` - Quick start guide
- `IMPLEMENTATION_SUMMARY.md` - Implementation details
- `agent_prompt_template.md` - Prompt template documentation

## Usage

See `README.md` or `QUICKSTART.md` for usage instructions.

## Development

```bash
make setup      # Set up development environment
make install    # Install dependencies
make lint       # Run linters
make format     # Format code
make check      # Run all checks
```
